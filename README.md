# Проект: Место

Сайт о любимых местах пользователя. Учебный проект Яндекс.Практикума. Сайт состоит из редактируемого блока с именем пользователя, разделом о себе и блока фотографий избранных мест.

### Технологии
HTML,
CSS,
Grid Layout,
Flexbox верстка,
JavaScript,
БЭМ
